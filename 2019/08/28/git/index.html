拉取远程分支到本地

  1. 首先把远程分支先拉到本地（情况：本地无远程分支dev,想拉取同名dev分支到本地）

  2. 在本地创建分支dev并切换到该分支，然后跟远程的origin地址上的dev分支关联起来

  3. 把远程某个分支上的内容都拉取到本地

  4. 再git add ,git commit ,git push代码

本地未切换分支修改代码

  • master分支代码提交（此次版本要修改的分支地址dev_20210101，未切换分支直接在master上修改代码）

  • 若要切换本地不存在的远程分支，先将远程git仓库里的指定分支拉取到本地

  • 合并master分支到dev_20210101

  • 提交dev_20210101

  • 切换到master分支,进行master分支回退版本

charles代理工具

安装

  1. 地址：http://www.charlesproxy.com/
  2. 设置破解key: https://www.zzzmode.com/mytools/charles/ 在此链接新建一个注册名，获取License key.
  3. “Help”->”register chales”添加你的name和key值。

抓取http

  1. 将Charles设置成系统代理，在启动Charles后，第一次Charles会请求你给它设置系统代理的权限，你可以输入登录密码 授予Charles该权限。若是你忽略了该授权，你需要选择菜单中的 “Proxy” –> “Mac OS X Proxy” 来将Charles设置成系统代理。之后，你就可以看到源源不断的网络请求出现在 Charles 的界面中。

  2. 但是Chrome 和 Firefox 浏览器默认并不使用系统的代理服务器设置，而 Charles 是通过将自己设置成代理服务器来完成封包截取的，所以在默认情况下无法截取 Chrome 和 Firefox 浏览器的网络通讯内容。选择菜单“Proxy”->”Proxy Setting”设置如下即可。当前设置的port是用于其他端服务器代理到本机的代理配置端口。

以上两步完成http抓包。

抓取https：

  1. 安装Charles 的 CA 证书：菜单【Help】–》【SSL Proxying】–》【Install Charles Root Certificate】，然后会跳转到mac的证书界面，在钥匙串中找到charles的证书，次证书默认是红色的，表示没有信任，点击完全信任即可。

  2. 打开【Proxy】–》【SSL Proxying Setting】菜单，点击add按钮，输入*，表示允许所有域名，这样就不用一个个手动添加了。

通过以上配置后，我们抓取的信息，就都可以看到明文了。

  • 注意事项：如果点开一个请求，里面都是unkown，而且左侧有个“🔒”的符号：说明未执行第II步。

设置代理（抓取本机内容包）:

  1. 打开 Proxy -> Recording Setting -> include ：设置你需要抓取的ip地址。
  2. 若是想抓取全部，则不需要任何设置，include所有选项都不选中即可。

设置其他电脑端代理（抓取其他电脑包，windows代理至mac）

  1. windows浏览器打开设置->Internet选项->连接->局域网设置

  2. mac会弹框允许代理，点击allow.

  3. 进行设置代理地址配置：打开 Proxy-Recoeding Setting-include ：设置你需要抓取的ip地址。
  4. 如果不需要代理了，需要把windows的代理配置去掉。
  • 备注： 其他端服务器代理到本机的代理配置端口：proxy-Http Proxy的Port端口。

设置手机代理（抓取手机上http的请求信息）

  1. 安装证书：打开Help->SSL Proxying->Install Charles…

  2. 打开手机WIFI，连接和电脑相同的WIFI，然后选中该无线网，添加http代理。

  3. 首先我们需要获取 Charles 运行所在电脑的 IP 地址：用ifconfig命令查看或者如下图打开菜单“Help”->Local IP Address查看。

  1. 在 iPhone 的 “ 设置 ”–>“ 无线局域网 ” 中，可以看到当前连接的 wifi 名，通过点击右边的详情键，可以看到当前连接上的 wifi 的详细信息，包括 IP 地址，子网掩码等信息。在其最底部有「HTTP 代理」一项，我们将其切换成手动，然后填上 上一步Charles 运行所在的电脑的 获取的IP，以及端口号 8888（proxy-Http Proxy的Port端口值）。

  2. 设置好之后，我们打开 iPhone 上的任意需要网络通讯的程序，就可以看到 Charles 弹出 iPhone 请求连接的确认菜单（如下图所示），点击 “Allow” 即可完成设置。

手机抓取https

  1. 访问chls.pro/ssl下载并安装证书
  2. 进入通用->关于本机->证书信任设置，将刚刚安装的证书设为信任即可
